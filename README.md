# EconTI — 经济学直觉视角测试 (Economic Intuition Lens)

> **"当你在观察身边世界的摩擦时，你潜意识里在套用哪一种经济学学理直觉？"**

EconTI 是一个极具美感、面向大众、经济学爱好者以及课堂展示的轻量化教育互动项目。它不带黑白道德色调，也不做严肃政治诊断，而是通过日常生活中的公共政策选择，呈现出人们看世界时底层的经济学解释框架与分析直觉。

---

## 🌟 核心特色与革新

1. **项目定位澄明**：从纠结于“你是哪个流派学家”的人格分类，升级为**“你看问题时，最先使用哪种解释框架 / 观察直觉（Economic Lenses）？”**，致力于帮助用户理解不同学说视角如何切入日常民生现象。
2. **六大核心经济学观察视角 (6 Core Lenses)**：
   * **价格机制型** (Price Mechanism Lens) — 审视价格、供给、需求和边际激励。
   * **宏观稳定型** (Macro Stabilization Lens) — 洞察系统运行、总需求周期与全局风险。
   * **行为现实型** (Behavioral Reality Lens) — 拥抱真实人类状态、偏误与巧妙的默认助推（Nudge）。
   * **分配能力型** (Equity & Capability Lens) — 关注弱者的起点平权、健康受教育尊严与可行能力。
   * **制度权力型** (Institutional Power Lens) — 剖析倾斜的议价话语、契约结构、垄断与风险甩包。
   * **创新演化型** (Evolutionary Innovation Lens) — 瞩目“创造性破坏”、颠覆式质变技术与企业家成长。
3. **社区治理隐藏指纹 (Ostrom Polycentrism)**：
   作为奥斯特罗姆公共治理理论的致敬，若用户在面临水能排放、共有住房公地两难中，倾向选择并支持本地利害人自治自设协商、协作决议，系统会单独唤醒轻量化的 **「社区协作与地方治理倾向」** 特征卡片，增强探究丰富度。
4. **10 道高信息量、无道德污名的现实权衡情境**：
   重设了所有场景（城市住房租赁、咖啡价格上涨、AI白领浪潮、骑手算法、名校学位竞争、重污染排放治理、挂号看病挤兑、商行坏账风波、阶层财富沉淀、社媒算法成瘾），将极具共鸣的微观与宏观机制包装成中性的、富有政策讨论感的均衡选项，去除诱导、排除说教。
5. **精准闭环计分机制**：
   优化了 React 异步多项累加和触发模型，确保第 10 题答完能百分百实时、绝对精准地计算最终匹配结果，支持两两契合的隐藏倾向列示，内置“全景辩证混合型”的自适应匹配拦截。

## 🛠️ 技术运行说明

本项目使用 **React (18+) + TypeScript + Vite + Tailwind CSS** 精炼构建。

### 1. 安装开发依赖
```bash
npm install
```

### 2. 启动本地开发服务端口（绑定 3000）
```bash
npm run dev
```

### 3. 生成生产环境打包文件
```bash
npm run build
```
