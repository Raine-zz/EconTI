# EconTI — Economic Intuition Type Indicator

> **经济学直觉视角测试 "当你在观察身边世界的摩擦时，你潜意识里在套用哪一种经济学学理直觉？"**

p.s. 该项目为作者所修课程 *人工智能与经济学前沿* 的期末项目，尝试通过 AI-assisted coding 将经济学解释框架转化为可交互的日常决策测验。本测验旨在以轻量方式呈现不同经济学视角，仅供学习交流与娱乐参考，不代表严肃学术分类、心理诊断或现实决策依据。

---

## 🌟 核心特色

1. **项目定位**：致力于帮助用户理解不同学说视角如何切入日常民生现象。
2. **六大核心经济学观察视角 (6 Core Lenses)**：
   * **价格机制型** (Price Mechanism Lens) — 审视价格、供给、需求和边际激励。
   * **宏观稳定型** (Macro Stabilization Lens) — 洞察系统运行、总需求周期与全局风险。
   * **行为现实型** (Behavioral Reality Lens) — 拥抱真实人类状态、偏误与巧妙的默认助推（Nudge）。
   * **分配能力型** (Equity & Capability Lens) — 关注弱者的起点平权、健康受教育尊严与可行能力。
   * **制度权力型** (Institutional Power Lens) — 剖析倾斜的议价话语、契约结构、垄断与风险甩包。
   * **创新演化型** (Evolutionary Innovation Lens) — 瞩目“创造性破坏”、颠覆式质变技术与企业家成长。
3. **倾社区协作与地方治理向(Ostrom Polycentrism)**：
  如果用户在环境治理、公共资源或社区事务等问题中，多次选择由本地参与者协商、制定规则并共同执行的方案，系统会在结果页中显示这一轻量副标签。它参考了奥斯特罗姆关于多中心治理与公共资源管理的思想，用来提示用户可能更重视地方知识、协作规则和共同治理。
4. **12 道现实经济权衡情境**：
   测验围绕城市住房租赁、咖啡价格上涨、AI 替代工作、骑手算法管理、名校学位竞争、污染治理、医疗资源紧张、银行坏账、财富积累、社交媒体成瘾，以及青年就业困难、产业结构转型等 12 个场景展开。每道题都尽量转化为中性的政策或生活选择，避免明显诱导和道德化表述，使用户在具体情境中体现自己的经济学直觉。
5. **可解释的计分与匹配机制**：
   系统会根据用户选择累积分数，识别主导经济学视角，并展示两个相近的隐藏倾向；当不同视角得分较为接近时，也会给出混合型结果，避免过度强行分类。

## 📊 评分体系与开发者深度诊断工具 (Scoring & Algorithmic Diagnostics)

为了确保测试系统具有极佳的学理对称性与极限制约度，我们设计并集成了底层的 **Developer Diagnostics Panel（开发者审计中心）**。

您可以点击页面最下方 (Footer) 区域的 **「评分算法诊断系统」** 进入该审计中心。该工具会一键拉起并运行以下四大诊断引擎：

1. **蒙特卡洛 1000 次随机模拟 (Monte Carlo Dispersion Checking)**：
   在 1000 次纯随机答题模拟负载下，检验六大主导直觉视角触发百分比的收敛对称度（现每视角潜在总分完美对齐 12-14 积分边界，彻底消除了以往价格机制/制度权力的溢出性压倒偏向，且奥斯特罗姆副特质唤醒率科学平衡在大约 25%）。
2. **极限定向还原度测试 (100% Recoverability Test)**：
   模拟 100% 对应视角的极端答题主体（每题必选该视角），检验算法是否能在 100% 极限极值下准确诊断还原主视角，反馈系统的异常失效抗干扰性。
3. **干扰刚度抗噪分析 (Sensitivity & Rigidity Audit)**：
   自动随机扰动 1 或 2 题，检测当主体由于误操作或偶然摇摆而偏离时，原主导诊断的稳健度（Held）或漂移趋势，确保算法具有优良的刚度自检属性。
4. **拟真人自适应概率主体仿真 (Probabilistic Archetype Agent Simulations)**：
   模拟四种典型的社科价值偏好（例如“弱者保护与社会福利偏好”、“自由竞争与市场纪律意识”等典型人群分类）进行高频行为决策仿真。统计在带有噪音（Epsilon noise）时的多维度期望得分，检验最终偏向与真实的理论预测是否具有严密的自恰性。

这使得 `EconTI` 的测评在面对大范围不同分布的用户流时，不仅能表现出优异的高美感视觉流，还具有学术经得起推敲的一致性刚度与逻辑自恰。
